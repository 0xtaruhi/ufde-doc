## 什么是 Rabbit？

Rabbit 是面向 FDP3P7 FPGA 开发板的虚拟元件平台，通过 USB 与板卡通信，并在桌面 GUI 中以交互式组件的方式展示/控制信号。

- 支持输入类组件（开关、按键、键盘、旋钮等）与输出类组件（LED、LCD、数码管、点阵等）
- 提供工程管理、位流下载、运行/停止控制、波形生成与查看集成
- 以 Qt 6/C++20 实现，使用 xmake 构建

参考仓库：[`0xtaruhi/Rabbit`](https://github.com/0xtaruhi/Rabbit)

## 下载地址
- 官方仓库：[`0xtaruhi/Rabbit`](https://github.com/0xtaruhi/Rabbit)
- 预编译包：[`Rabbit-Precompiled`](https://github.com/0xtaruhi/Rabbit/releases/tag/v1.0.1)预编译的版本可能没有最新的功能，并有潜在的 bug。推荐从官方仓库编译。

<Callout type="warn">
  Linux 系统下需要以管理员权限运行。否则可能出现开发板无法连接的问题。
</Callout>

要快速开始，可以下载预生成的[比特流](/files/Dino.bit)与[约束文件](/files/Dino_cons.xml)，按照[新建项目与设置](./rabbit/project)章节进行操作。
该示例提供了一个简单的恐龙游戏，可以参考[界面与页面导航](./rabbit/pages)章节进行操作。
